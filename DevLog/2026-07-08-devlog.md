### 4. 플레이어 애니메이션 추가

- 타이틀 화면 UI를 구성했다.
- 로그인 화면 UI를 구성했다.
- 회원가입 화면 UI를 구성했다.
- 각 화면에 필요한 버튼, 입력 필드, 텍스트 오브젝트를 배치했다.
- UI 오브젝트 이름을 역할이 드러나도록 정리했다.

### 6. UI 화면 전환 기능 구현

- 타이틀 화면에서 로그인 화면으로 이동할 수 있도록 구현했다.
- 로그인 화면에서 회원가입 화면으로 이동할 수 있도록 구현했다.
- 회원가입 화면에서 로그인 화면으로 돌아올 수 있도록 구현했다.
- 세 개의 화면이 서로 연결되도록 기본 UI 전환 로직을 구현했다.

## 문제 상황

### 1. Unity `Library` 폴더가 Git 변경 사항에 표시됨

Unity 프로젝트가 저장소 루트가 아니라 `EndlessDay/` 하위 폴더에 위치해 있어, 기존 `.gitignore` 규칙이 `Library` 폴더를 제대로 제외하지 못했다.

## 해결 방법

### 1. `.gitignore` 규칙 수정

Unity 자동 생성 폴더가 하위 폴더에서도 제외되도록 `.gitignore` 규칙을 수정했다.

```gitignore
**/[Ll]ibrary/
**/[Tt]emp/
**/[Oo]bj/
**/[Bb]uild/
**/[Bb]uilds/
**/[Ll]ogs/
**/[Uu]ser[Ss]ettings/
```
