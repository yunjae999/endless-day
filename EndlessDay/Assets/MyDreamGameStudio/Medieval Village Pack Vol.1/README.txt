Installation

1. Download the Medieval Fantasy Village Pack from the asset store.
2. Import the package into your Unity project.
3. Extract the contents to a folder of your choice.

Usage

1. Browse the prefabs folder to find the desired prop.
2. Drag and drop the prefab into your scene.
3. Customize the models's materials and settings as needed.

Technical Details

- Number of Models: 35
- Texture Styles: Aged
- Number of Prefabs: 35
- Texture Resolution: 2048x2048
- Format: FBX, Prefabs, Textures
- Compatibility: Unity 2018.4 or later, Compatible with URP, and Built-in Render Pipeline
- Platforms: Windows, Mac, Linux, iOS, Android, WebGL

Material Upgrade

To use materials with a different Render Pipeline, please follow these steps:

1. Find desired Render Pipeline package in asset folder.
2. Click on and import.
